## 🚀 How to Deploy (GitHub Pages — Free)

### Step 1: Create GitHub Account
- Visit [https://github.com](https://github.com) → Sign up
- Suggested email: `saimuralipadala143@gmail.com`

### Step 2: Create New Repository
1. Click **+** (top right) → **New repository**
2. Repository name: `saimurali-portfolio`
3. Visibility: ✅ **Public**
4. Click **Create repository**

### Step 3: Upload Files
1. In your new repo → click **Add file** → **Upload files**
2. Upload both:
   - `index.html`
   - `README.md`
3. Click **Commit changes**

### Step 4: Enable GitHub Pages
1. Go to **Settings** tab → scroll to **Pages**
2. Under *Source* → select **Deploy from a branch**
3. Branch: `main` | Folder: `/ (root)`
4. Click **Save**

### Step 5: Site is Live! 🎉
Wait 1–2 minutes. Your portfolio will be live at:
```
https://YOUR-USERNAME.github.io/saimurali-portfolio/
```
